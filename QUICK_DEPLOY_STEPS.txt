DEPLOY TO SERVER 4.230.0.0
========================================

STEP 1: TRANSFER TO SERVER
---------------------------
Replace USERNAME with your SSH username:

  scp /home/conorhan/cafe/saejaedang-new-design.tar.gz USERNAME@4.230.0.0:~/

STEP 2: SSH TO SERVER
---------------------
  ssh USERNAME@4.230.0.0

STEP 3: EXTRACT & DEPLOY
-------------------------
Run these commands on the server:

  # Extract
  mkdir -p ~/saejaedang
  cd ~/saejaedang
  tar -xzf ~/saejaedang-new-design.tar.gz

  # Install & Build Frontend
  cd ~/saejaedang/frontend-new
  npm install
  npm run build

  # Stop old frontend
  pm2 stop saejaedang-frontend 2>/dev/null || true
  pm2 delete saejaedang-frontend 2>/dev/null || true

  # Start new frontend
  npm install -g serve
  pm2 start "serve -s build -l 3000" --name saejaedang-frontend
  pm2 save

  # Check status
  pm2 status

========================================
NGINX SETUP (Optional - Port 80)
========================================

If you want to serve on port 80 with Nginx:

  sudo apt-get install -y nginx

  # Create config (replace USERNAME with your username)
  sudo nano /etc/nginx/sites-available/saejaedang

  # Paste this config:
  server {
      listen 80;
      server_name 4.230.0.0;
      root /home/USERNAME/saejaedang/frontend-new/build;
      index index.html;
      location / {
          try_files $uri $uri/ /index.html;
      }
      location /api {
          proxy_pass http://localhost:5000;
      }
  }

  # Enable
  sudo ln -s /etc/nginx/sites-available/saejaedang /etc/nginx/sites-enabled/
  sudo rm /etc/nginx/sites-enabled/default
  sudo nginx -t
  sudo systemctl reload nginx

  # Open firewall
  sudo ufw allow 80/tcp
